Perlin's noise

This graphs were generated using the P5 javascript library. It shows how the perlin's noise algorithm generates an interpolation between random values so that they are related (stop being random - each value has an impact on those around it). This algorithm is usually used to generate realistic terrains and cool patterns. It was invented by Ken Perlin to generate the textures for the movie Tron (1997).

Try it yourself!



In this side you can see a set of particles that start in a random position and are influenced by a vector field. Both the initial direction and the movement of these vectors is defined with the Perlin's noise algorithm. It kind of remembers me of murmurations. Images like showed can also be created with this algorithm.
